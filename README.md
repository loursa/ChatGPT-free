## **5. 安全性与隐私保护**

在使用 ChatGPT 中文版和官网时，请注意以下事项：  
- **敏感信息保护：**  
  避免在对话中输入身份证号、银行卡号等个人敏感信息。  
- **网站安全性：**  
  确保访问的网站使用 HTTPS 加密协议，避免数据被窃取。  
- **隐私协议：**  
  优先选择列出隐私条款和用户协议的站点，确保数据安全。
